# Dashboard
E-commerce Sales Dashboard
A Dynamic Interactive data visualization tool to explore E-commerce sales dashboard focusing on quarter data based on states

Purpose
Created interactive dashboard to track and analyze online sales data
Used complex parameters to drill down in worksheet and customization using filters and slicers
Created connections, join new tables, calculations to manipulate data and enable user driven parameters for visualizations
Used different types of customized visualization (bar chart, pie chart, donut chart, clustered bar chart, scatter chart, line chart, area chart, map, slicers, etc)

Tech stack
Business Intelligence: Microsoft Power BI
• Data Transformation: Power Query
• Data Modeling: Relationships, Calculated Columns
• Analytics Language: DAX (Data Analysis Expressions)
• Data Visualization: Interactive Dashboards, KPI Cards, Sales & Profit Analysis
• Data Handling: Data Cleaning, Data Transformation
• Tools: Microsoft Excel, Power BI Desktop, Power BI Service
• Time Intelligence: Date Table, Monthly & Quarterly Sales Analysis

Source 
Extract from kaggle

Highlights
Developed an interactive Sales Analytics Dashboard in Power BI to analyze sales performance, profit trends, and customer purchasing patterns.

Performed data cleaning and transformation using Power Query to prepare raw sales data for analysis.

• Built a data model with relationships between Orders, Details, and Date tables to enable efficient analysis.

• Created DAX measures and calculated columns to compute KPIs such as Total Sales, Profit, Quantity Sold, and Average Order Value (AOV).

• Designed interactive visualizations and filters to analyze sales by Category, Sub-Category, State, City, and Payment Mode.
